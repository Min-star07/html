Git Notes

1. What is Git?

Git is a distributed version control system that allows multiple users to collaborate on projects without conflicts.

2. Basic Commands and Working with Remote Repositories

Essential commands for Git include:

  • git init - Initializes a new Git repository.
  • git clone <repository-url> - Copies a remote repository locally.
  • git add <file-name> - Stages changes for commit.
  • git commit -m "message" - Commits staged changes with a message.
  • git push - Pushes local commits to the remote repository.

To connect to a GitHub repository, follow these steps:

  • git init
  • git remote add origin <repository-url>
  • git add .
  • git commit -m "Initial commit"
  • git push -u origin master

This initializes a repository, sets the remote URL, stages files, commits them, and pushes to GitHub.

3. Branching and Merging

Branching allows parallel development. Use

  • git branch <branch-name> to create a branch,
  • git checkout <branch-name> to switch, and
  • git merge <branch-name> to combine branches.
